# Важные папки Windows для разработчика

```blogEnginePageDate
17 июля 2026
```

В Windows многие системные и пользовательские каталоги можно открывать через переменные окружения. Достаточно вставить
нужный путь в адресную строку Проводника или в окно `Win + R`.

## Профиль пользователя

| Путь                       | Назначение                              |
|----------------------------|-----------------------------------------|
| `%USERPROFILE%`            | Корневая папка текущего пользователя    |
| `%USERPROFILE%\Desktop`    | Файлы рабочего стола                    |
| `%USERPROFILE%\Downloads`  | Загруженные файлы                       |
| `%USERPROFILE%\.ssh`       | SSH-ключи и конфигурация SSH            |
| `%USERPROFILE%\.m2`        | Настройки и локальный репозиторий Maven |
| `%USERPROFILE%\.gradle`    | Настройки и кеш Gradle                  |
| `%USERPROFILE%\.npm`       | Кеш npm                                 |
| `%USERPROFILE%\.npmrc`     | Настройки npm                           |
| `%USERPROFILE%\.gitconfig` | Глобальная конфигурация Git             |
| `%USERPROFILE%\.cursor`    | Пользовательские настройки Cursor       |

## Системные файлы Windows

| Путь                            | Назначение                                    |
|---------------------------------|-----------------------------------------------|
| `%WINDIR%`                      | Системная папка Windows, обычно `C:\Windows`  |
| `%WINDIR%\System32\drivers\etc` | Системные сетевые файлы, включая файл `hosts` |

## Автозагрузка

| Путь                                                          | Назначение                             |
|---------------------------------------------------------------|----------------------------------------|
| `%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup`     | Автозагрузка для текущего пользователя |
| `%PROGRAMDATA%\Microsoft\Windows\Start Menu\Programs\Startup` | Автозагрузка для всех пользователей    |

Ярлык, помещённый в одну из этих папок, будет запускаться при входе пользователя в Windows.

## Меню «Пуск» и ярлыки

| Путь                                                 | Назначение                                      |
|------------------------------------------------------|-------------------------------------------------|
| `%APPDATA%\Microsoft\Windows\Start Menu`             | Ярлыки меню «Пуск» текущего пользователя        |
| `%PROGRAMDATA%\Microsoft\Windows\Start Menu`         | Общие ярлыки меню «Пуск» для всех пользователей |
| `%APPDATA%\Microsoft\Internet Explorer\Quick Launch` | Ярлыки быстрого запуска и панели задач          |
